Primi script

Per la scrittura di programmi in javascript si consiglia innanzitutto di configurare caratteristiche generali quali la codifica di caratteri o il carattere di fine riga, utilizzando strumenti quali EditorConfig.

Inserimento script in pagina

Per inserire uno script in una pagina web esistono diverse possibilità:

  • inserire uno script inline ovvero nei tag HTML che ne permettono la possibilità;
  • inserire uno script direttamente nella pagina HTML, utilizzando il tag script;
  • inserire un collegamento ad un file javascript esterno, utilizzando il tag script;

Per inserire il tag inline, lo si può allegare ai tag HTML che lo prevedono; ad esempio al tag button:

    
        <input type="button" onclick="alert('Script contenuto inline');">

Quando l'utente clicca sul pulsante, si attiva l'azione "onclick" e quindi si visualizza il messaggio.

Per inserire uno script direttamente nella pagina web, si usa il tag script per includere il codice. Di seguito un esempio:

    
        <script>
            alert("script contenuto nella pagina web");
        </script>

Per inserire un collegamento ad un file esterno, si inserisce il tag script con l'attributo src che contiene il collegamento al file javascript esterno. Di seguito un esempio:

    
        <script src="file.js"></script>

Gli script vengono eseguiti man mano che la pagina javascript viene "letta" dal browser e mostrata all'utente. Quindi conta l'ordine in cui il codice è scritto nella pagina web.

Uso della console

Lo strumento che più aiuta nella scrittura di codice javascript è la console del browser, che appare quando si preme il tasto "F12" o quando si clicca con il seconda tasto del mouse sulla pagina web e si sceglie la voce "Ispeziona".

La console aiuta lo sviluppo poichè lo sviluppatore può scrivere sulla console tutte le informazioni di cui ha bisogno.

Nella console è possibile scrivere informazioni, allarmi ed errori attraverso le istruzioni console.info(), console.warn() e console.error():

    
        console.info("informazione");
        console.warn("allarme");
        console.error("errore");

E' possibile anche creare gruppi di messaggi, utilizzando le istruzioni console.group() e console.groupEnd():

    
        console.group("gruppo di informazioni");
        console.info("informazione 1");
        console.info("informazione 2");
        console.warn("allarme 1");
        console.groupEnd();

La scrittura di una tabella nella console, attraverso l'istruzione console.table(), è particolarmente utile poichè permette di visualizzare array e matrici in un formato molto comprensibile:

    
        let vettore = ['uno', 'due', 'tre'];
        console.table();

Utilissima pure la scrittura in console del tempo di esecuzione intercorso tra due punti dello script, indicati dalle istruzioni console.time() e console.timeEnd():

    
        console.time("tempo impiegato:");

        // istruzioni

        console.timeEnd("tempo impiegato:");

In output ci sarà la scritta "tempo impiegato: 12.014 ms".

Per stampare la struttura di un oggetto si usa l'utilissima istruzione console.dir(), come nel seguente esempio

    
        let obj = {nome: "Pippo", cognome: "Pluto"};
        console.dir(obj);
